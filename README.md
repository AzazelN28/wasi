# WASI

## Installation

To install WASI SDK run:

```sh
./download.sh
```

Now you should install [wasmer](https://docs.wasmer.io/ecosystem/wasmer/getting-started) with:

```sh
curl https://get.wasmer.io -sSfL | sh
```

Once installed you should have a program called `wasmer`:

## Compilation

```sh
./setup.sh
make
```
